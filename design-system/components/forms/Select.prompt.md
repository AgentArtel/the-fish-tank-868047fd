Styled native `<select>` with custom chevron and coral focus ring. Pass `<option>`s as children.

```jsx
<Select label="Category" defaultValue="all">
  <option value="all">All categories</option>
  <option value="fish">Saltwater Fish</option>
  <option value="coral">Corals</option>
  <option value="invert">Inverts</option>
</Select>
```
