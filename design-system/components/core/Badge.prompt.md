Small pill label for categories, item types, and live-stock status — `soft` tone for filters/tags, `solid` for emphasis, `dot` for stock state.

```jsx
<Badge tone="ocean">Saltwater Fish</Badge>
<Badge tone="success" dot>In stock</Badge>
<Badge tone="gold" variant="solid">New arrival</Badge>
<Badge tone="neutral" variant="outline">SPS</Badge>
```

Tones: `blue`, `cyan`, `gold` (yellow), `ocean` (deep royal), `neutral`, `success`, `warning`, `danger`. Variants: `soft` (default, tinted), `solid`, `outline`. Add `dot` for a leading status dot (great for the live catalog).
